# Smart Lender

An AI-powered Loan Approval Prediction System using Machine Learning and Flask.

## Features
- Loan Approval Prediction
- Random Forest Machine Learning Model
- Flask Web Application
- Responsive User Interface
- Confidence Score
- Splash Screen

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS
- JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
```

## Project Structure

- app.py
- train_model.py
- templates/
- static/
- models/
- dataset/
